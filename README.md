# Smart Page Translator

仓库：[https://github.com/843773493/smartpagetranslator](https://github.com/843773493/smartpagetranslator)

在 VS Code 编辑器标题栏添加翻译按钮，可将当前文本从英文翻译为中文。

## 功能特性

- 🌐 在编辑器标题栏添加翻译按钮
- 📝 支持将整个编辑器内容从英文翻译为中文
- 🔄 使用 Bing 翻译 API
- 💬 实时提示翻译状态

## 使用方法

1. 安装插件
2. 打开要翻译的文件
3. 点击编辑器标题栏右侧的翻译按钮（或使用命令 `Translate to Chinese`）
4. 等待翻译完成，文本将被替换为中文

## 依赖

- `bing-translate-api` - 用于文本翻译

## 开发

```bash
npm install
npm run compile
```

启动调试：按 F5 启动 VS Code 插件的调试会话
